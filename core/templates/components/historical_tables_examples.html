Component Features

Key Features

  • Two Theme Options: Golden (luxury) and Minimal (clean) styling
  • Responsive Design: Adapts to mobile, tablet, and desktop screens
  • Interactive Elements: Click to copy values, sortable columns, hover effects
  • Flexible Data Support: Works with arrays or dictionary objects
  • Export Functionality: Built-in CSV export with clean data
  • Pagination Support: Server-side pagination with navigation controls
  • Customizable Actions: Configurable action buttons and callbacks
  • Empty State Handling: Professional no-data display
  • Color-coded Values: Positive/negative changes with appropriate colors
  • Accessibility: Proper ARIA labels and keyboard navigation
